# **App Name**: Reflecto

## Core Features:

- Personal Profile Display: Display the developer's professional details including name, role, location, and a profile picture.
- Skill Showcase: List technical skills and expertise in web development, cybersecurity, and software development.
- Projects Section: Showcase a portfolio of projects with descriptions and links to live demos or GitHub repositories.
- Contact Information: Provide a contact form and display social media links, like LinkedIn, Twitter, and email for easy contact.
- Testimonials Carousel: Showcase testimonials and recommendations to build credibility. Source for testimonials can be user input. To minimize cold starts a single global prompt is used as a tool to determine the persona for each of these fictional users and their recommendation message.
- Age Calculator: Automatically calculates the developer's age based on their birthday.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB), echoing the clear and modern iOS aesthetic.
- Background color: Very light grey (#F0F0F0), mimicking iOS frosted glass effect, adding depth without distraction.
- Accent color: Teal (#008080), analogous to sky blue, creating contrast for interactive elements.
- Body and headline font: 'Inter' sans-serif for clear and modern readability, maintaining a professional yet approachable feel.
- Use simplistic, custom icons inspired by the iOS app icons, providing clear visual cues for navigation. These will be reminiscent of glass morphism.
- Design a mobile-first, grid-based layout, inspired by iOS app arrangements, ensuring responsiveness and a familiar user experience across devices.
- Employ subtle transition animations, like iOS's smooth modal transitions, for a fluid and engaging browsing experience.
